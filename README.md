# 目标是获得一个能够代替人类陪伴你的对话者

## 技术特点
+ 根据LCCC数据集的中的**词频**构造词典，且提供**剪枝**以控制词典大小
  + 虽然有 bert-Chinese，但 LCCC 语料已经以“词”为最小元分好了，如果用“字”典岂不是浪费？
+ 翻译训练集和对话训练集不同，翻译只存在单轮（一句对一句），而对话数据集包含多轮对话
  + 将多轮对话拆分为连续的单轮对话
+ 对矩阵相乘和广播的原理进行详细的拆解和解释
+ 抽象出掩码
